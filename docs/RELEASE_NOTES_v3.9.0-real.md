- Multi-panel operator cockpit under `/v3/cockpit`.
- Saved cockpit layouts and review focus modes.
- Safe keyboard shortcut manifest and UI help.
- Safe command-palette manifest and API with forbidden live-mutation rejection.
- Side-by-side task/source/review context.
- Lightweight dependency-chain visualization.
- Cockpit JSON, Markdown, focus-mode, command-palette, shortcut, and CSV exports.
- Command center, search, graph, workflow, demo, screenshot, validation, and test integration.
