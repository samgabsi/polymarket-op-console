# Release Notes — v3.9.0-real

v3.9.0-real adds the **Guided Operator Workspace, Interactive Review Flows, and Task Dependency Intelligence** layer.

## Highlights

- `/v3/workspace` guided operator workspace and review launcher.
- Daily guided review, weekly guided review, and task triage flows.
- Task dependencies, blocked-task review, dependency exports, and dependency graph nodes.
- Source-context previews before task creation.
- Saved task views for due, overdue, blocked, source-specific, and safety-class-specific work.
- Guided review packets and workspace exports in JSON, Markdown, and CSV where practical.
- Command center, task planner, search, graph, workflow, demo, validation, screenshot, and documentation integration.

## Safety

Guided reviews, dependencies, source previews, saved views, review packets, and workspace exports do not place orders, cancel orders, approve trades, sign transactions, arm live trading, bypass backend gates, or provide financial advice.
