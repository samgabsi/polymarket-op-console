# Release Notes — v3.3.0-real

v3.3.0-real adds the Operator Analytics, Performance Learning, and Decision Quality Intelligence Layer.

## Added

- `/v3/analytics` workspace and analytics subroutes.
- `app/live_v3_analytics.py` local-first analytics engine.
- Decision quality metrics.
- Thesis quality metrics.
- Evidence usefulness metrics.
- Alert usefulness metrics.
- Governance discipline analytics.
- Portfolio/risk process analytics.
- Confidence calibration buckets with sample size and unknown counts.
- Mistake pattern analytics.
- Strength pattern analytics.
- Review follow-through analytics.
- Learning report generator.
- Analytics JSON, Markdown, and CSV exports.
- Analytics integration with command center, search, graph, and v3 workflow packets.
- Analytics-friendly demo fixture updates.

## Safety

Analytics are descriptive workflow guidance only. They do not place, cancel, sign, approve, arm, or bypass live-trading gates.
