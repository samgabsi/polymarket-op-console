# v3 Operator Intelligence OS Guide — v3.3.0-real

v3.3.0-real keeps the v3 Operator Intelligence OS local-first and human-controlled while redesigning the operator interface for usability, speed, and clarity.

## What v3 is

A unified command center linking live controls, strategy, research, monitoring, portfolio, governance, data integrity, global search, decision graph, read-only workflows, reports, and analytics.

## What v3 is not

v3 is not autonomous trading, financial advice, hidden execution, or a replacement for backend safety gates.

## v3.3 improvements

- Redesigned command center.
- Grouped navigation.
- Design system for status badges, cards, tables, filters, reports, and warning panels.
- Improved search/graph/workflow/report layouts.
- Stronger safety visibility.
- Better responsive and accessibility baseline.

## Safety boundary

v3 outputs are operator-guidance drafts. They do not place, cancel, approve, sign, or arm live orders.
