# Release Notes — v3.6.0-real

Adds the Read-Only Collection Scheduler, Dataset Freshness Planner, and Operator Notification Layer. The release introduces freshness policies, queued manual collection jobs, readiness reports, stale dataset/snapshot findings, local notifications, freshness exports, and command-center/search/graph/analytics/simulation/workflow integration. No orders are placed or cancelled and scheduler behavior is disabled by default unless explicitly enabled.
