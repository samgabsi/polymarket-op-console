# v3 Operator Analytics Guide — v3.3.0-real

v3.3.0-real preserves the v3.2 local-first analytics layer and presents it through a cleaner, faster, more readable analytics UI.

Analytics remain descriptive and non-autonomous. They summarize decision quality, thesis quality, evidence usefulness, alert usefulness, governance discipline, portfolio/risk process, confidence calibration, mistake patterns, strength patterns, and review follow-through.

Analytics are not predictions, financial advice, or execution authorization. Unknown outcomes remain unknown.
