# Release Notes — v3.3.0-real

v3.3.0-real is the first post-v3.0 release-candidate hardening pass. It refines the v3 Operator Intelligence OS without changing the live-trading safety model.

## Highlights

- Polished `/v3` command center layout and safety posture grouping.
- Added local search filter metadata and improved result presentation.
- Added graph filter metadata and cleaner object relationship exploration.
- Added workflow output templates for all major v3 read-only packets and briefs.
- Added safe fake demo fixture support for visual QA and screenshots.
- Added visual QA checklist and screenshot helper dry-run script.
- Added v3 release validation harness.
- Updated v3 docs, README, tests, and release checklist.

## Safety

v3.2 workflows, packets, briefs, graph findings, search results, and demo data are operator guidance only. They do not place orders, cancel orders, sign orders, approve orders, arm live trading, or bypass backend gates.
