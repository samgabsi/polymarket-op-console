# Release Notes — v3.0.0-real

v3.0.0-real introduces the Unified Operator Intelligence OS: a local-first command center that connects live readiness, strategy, research, monitoring, portfolio exposure, governance, data integrity, audit trails, and read-only workflow packets.

## Highlights

- New `/v3` command center and v3 navigation namespace.
- Global local search across strategy, research, monitoring, portfolio, governance, data health, and audit records.
- Local decision graph / object graph linking markets, theses, evidence, sources, alerts, exposure, scenarios, governance, tickets, orders, and audit events.
- Read-only workflow orchestrator for operator-triggered intelligence packets.
- Pre-trade intelligence packets.
- Market intelligence briefs.
- Thesis health reports.
- Portfolio risk briefs.
- Daily and weekly operator review packets.
- Missing-prerequisite and conflict detection.
- v3 settings boundary with AI/model assistance disabled by default.
- v2-to-v3 migration documentation.

## Safety

v3 intelligence outputs are workflow guidance only. They do not place orders, cancel orders, approve orders, sign orders, arm live trading, or bypass backend safety gates.
