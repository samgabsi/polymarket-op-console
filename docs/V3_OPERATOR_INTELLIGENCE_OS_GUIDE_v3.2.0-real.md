# v3 Operator Intelligence OS Guide — v3.3.0-real

v3.2 keeps the v3 Operator Intelligence OS intact and adds a learning-oriented analytics workspace.

Core v3 areas:

- Command Center
- Global Search
- Decision Graph
- Read-only Workflow Orchestrator
- Packets and Briefs
- Operator Analytics
- Settings and Documentation

Analytics integrate into the command center, global search, graph, and workflow outputs. All v3 intelligence remains read-only/operator-guidance unless the operator uses existing guarded live-trading flows separately.

Safety boundary: v3 intelligence, analytics, reports, and packets are not orders, not approvals, and not financial advice.
