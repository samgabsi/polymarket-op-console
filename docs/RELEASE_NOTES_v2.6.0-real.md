# Release Notes — v2.6.0-real

## Summary

v2.6.0-real adds the Market Monitoring and Alert Workflow Layer.

## Highlights

- New `/v2-live/monitoring` workspace
- Alert rule builder
- Price/spread/liquidity/status/watchlist/thesis/evidence/readiness alert categories
- Manual rule evaluation
- Active-alert notification center
- Alert acknowledgement and snooze
- Monitoring audit events
- JSON, Markdown, and CSV exports
- Documentation and tests for no autonomous execution

## Safety

No real order placement, cancellation, signing, approval, or live arming is performed by monitoring. Alerts are workflow guidance only.
