Configuration & Environment Setup Console

Turn supported environment, runtime, training, data, paper, live-readiness, and LAN options into guided controls. Text entry is reserved for paths, URLs, labels, hostnames, secrets, and other values that cannot safely be constrained.

Safety posture: This console writes supported .env keys only after validation and diff preview. It never executes shell commands, never mutates venv packages, never exposes full secrets, and never bypasses manual review or live-trading gates. Training outputs remain manual_review_only=true and can_live_trade=false.
