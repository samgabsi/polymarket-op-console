# Polymarket Gamma Starter

This repository contains a Polymarket Gamma starter template/boilerplate. It includes the zip archive (`polymarket-gamma-starter-v1.0.1-real.zip`) and an unzipped copy of the code.

## Getting started

1. Clone the repo:
   ```bash
   git clone https://github.com/samgabsi/polymarket-gamma-starter.git
   cd polymarket-gamma-starter
   ```

2. If you need to unzip the archive:
   ```bash
   unzip polymarket-gamma-starter-v1.0.1-real.zip -d ./polymarket-gamma-starter
   ```

3. Install dependencies and run (depending on your stack):

   **Node.js example**
   ```bash
   npm install
   npm run start
   ```

   **Python example**
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   python -m pip install -r requirements.txt
   python main.py
   ```
